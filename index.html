Projetos desenvolvidos

Countdown

Desafio da trilha Discover da Rocketseat, onde o objetivo era criar uma página de coming soon (em breve) para o lançamento de um site, produto ou serviço, e na página deveria ter um countdown timer (timer de contagem regressiva) informando quanto tempo falta para o lançamento. Utilizando HTML, CSS e Javscript.

NLW ESports

Projeto desenvolvido durante a NLW (Next Level Week) da Rocketseat. O objetivo era criar uma página de jogos e streamers favoritos. Utilizando apenas HTML e CSS.

Pokédex

Projeto desenvolvido com o objetivo de aprender mais sobre consumo de APIs e ReactJS. O intuito era que a API buscasse os dados que o usuário passasse e o ReactJS criasse um componente a partir do retorno da API. Neste projeto foram utilizadas as tecnologias: Figma, HTML, CSS, ReactJS, GIT e API Rest.

Rocket NFT

Uma landing page de NFT (non-fungible tokens) é uma página que visa apresentar e promover um conjunto de tokens únicos e inestimáveis ​​que representam propriedade digital de alguma forma. O projeto foi realizado utilizando somente as tecnologias HTML e CSS.

Netflix 90ths

O projeto consiste em criar uma réplica da interface da Netflix, utilizando o pacote "styled components" e "react 95" para dar um visual retro ao projeto. A aplicação exibe uma lista de filmes e séries, permitindo ao usuário navegar sobre cada título. O React 95 fornece componentes visuais já prontos para usar, como botões, janelas e cabeçalhos.

Habits

A aplicação é uma plataforma web e mobile para controlar hábitos diários. Foram utilizadas as tecnologias, Prisma para o banco de dados, ReactJS para a versão web e React Native para a versão mobile, a aplicação oferece uma interface intuitiva e fácil de usar para que os usuários possam gerenciar seus hábitos de forma eficiente.
